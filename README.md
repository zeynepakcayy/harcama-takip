# Harcama Takip Uygulaması

Bulut bilişim dersi dönem projesi olarak geliştirilen, kullanıcıların günlük harcamalarını kategori bazlı takip edebildiği iki katmanlı (web API + frontend) bir web uygulamasıdır.

## Özellikler
- Kullanıcı kayıt ve giriş (JWT authentication)
- Harcama ekleme (miktar + kategori)
- Kategori bazlı pasta grafik (pie chart)
- Toplam harcama görüntüleme
- Harcama listesi ve silme

## Teknolojiler
- **Backend:** Node.js + Express
- **Frontend:** React
- **Veritabanı:** (sonra eklenecek)
- **Bulut:** AWS
